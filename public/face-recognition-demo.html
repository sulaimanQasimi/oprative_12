Face Recognition Demo

Setup Instructions:

  • Download face-api.js models from GitHub
  • Place the models in /public/models/ directory
  • Required models: ssd_mobilenetv1_model, face_landmark_68_model, face_recognition_model, face_expression_model, age_gender_model
  • Get your API token from Laravel Sanctum authentication
  • Make sure your camera is connected and permissions are granted
